📥 Installation

Prêt en 2 minutes

Aucune compétence technique requise. Suis ces 4 étapes simples.

1

Télécharger le ZIP

Récupère la dernière version depuis les releases GitHub et décompresse le dossier.

2

Ouvrir les extensions

Va sur chrome://extensions/ ou about:debugging selon ton navigateur.

3

Mode développeur

Active le mode développeur en haut à droite (Chrome/Edge) ou via "Ce Firefox".

4

Charger le dossier

Clique sur "Charger l'extension non empaquetée" et sélectionne le dossier décompressé.

FAQ

Questions fréquentes

PixHelp est-il gratuit ? ▾
Oui, PixHelp est entièrement gratuit et open-source. Pas d'abonnement, pas de version premium. Le code source est accessible sur GitHub.
L'extension fonctionne-t-elle sur Firefox ? ▾
Oui ! PixHelp est compatible avec Chrome, Microsoft Edge et Firefox grâce à Manifest V3. Chaque navigateur a sa propre procédure détaillée dans la section installation.
Est-ce que PixHelp collecte mes données ? ▾
Non. PixHelp n'a aucun serveur ni base de données. Tout fonctionne localement dans ton navigateur. Aucune donnée ne quitte ton appareil. Sauf avec l'utilisation de PixHelp Prof.
L'extension se désactive-t-elle pendant les certifications ? ▾
Oui, c'est automatique. Sur toutes les pages /certifications de Pix, PixHelp détecte le contexte, se désactive et floute la section avec un avertissement visible.
Comment contribuer ou signaler un bug ? ▾
Ouvre une issue ou une pull request sur GitHub. Les contributions sont les bienvenues !
