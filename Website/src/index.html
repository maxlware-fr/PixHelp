✨ Fonctionnalités

Tout ce dont tu as besoin

Une extension légère et efficace, pensée pour t'accompagner au quotidien sur Pix.

✅

Aide sur les questions

PixHelp t'aide à trouver et comprendre les réponses aux questions Pix de manière rapide et efficace. Plus de blocage sur les exercices difficiles.

⚙️

Paramètres personnalisables

Active ou désactive l'extension en un seul clic depuis la barre d'outils. Interface épurée avec onglets Infos et Options.

🚫

Protection certification

Sur /certifications, l'extension se désactive automatiquement et floute la section pour éviter toute tricherie en conditions officielles.

🔒

Zéro collecte de données

PixHelp ne collecte aucune donnée personnelle. Tout fonctionne localement dans ton navigateur, sans envoi de requêtes externes.

⚡

Léger & rapide

Extension minimaliste qui n'alourdit pas ton navigateur. Conçue pour ne pas ralentir le chargement des pages Pix.

🛠️

Open-source

Le code source est entièrement disponible sur GitHub. Tu peux inspecter, forker et contribuer librement au projet.

📥 Installation

Prêt en 2 minutes

Aucune compétence technique requise. Suis ces étapes simples.

1

Télécharger le ZIP

Récupère la dernière version depuis les releases GitHub et décompresse le dossier.

2

Ouvrir les extensions

Accède à chrome://extensions/ ou about:debugging selon ton navigateur.

3

Activer le mode dev

Active le mode développeur (Chrome/Edge) ou va dans "Ce Firefox" (Firefox).

4

Charger le dossier

Clique sur "Charger l'extension non empaquetée" et sélectionne le dossier téléchargé.

🛡️ Éthique

Conçu de manière responsable

PixHelp est pensé pour l'apprentissage, pas pour tricher. Des garde-fous automatiques protègent l'intégrité des certifications officielles.

🚫 Blocage certifications

Sur toutes les pages /certifications, PixHelp se désactive automatiquement. Le contenu est flouté et un message d'avertissement s'affiche.

📚 Pour l'apprentissage

L'outil est conçu pour comprendre et progresser sur les compétences numériques, pas pour contourner les évaluations officielles.

⚠️ Extension non officielle

PixHelp n'est pas affiliée à Pix. Utilise-la dans le cadre autorisé par ton établissement scolaire.

❓ FAQ

Questions fréquentes

PixHelp est-il gratuit ? ▾
Oui, PixHelp est entièrement gratuit et open-source. Tu peux consulter le code source sur GitHub à tout moment.
L'extension fonctionne-t-elle sur Firefox ? ▾
Oui ! PixHelp supporte Chrome, Microsoft Edge et Firefox. Chaque navigateur a sa propre procédure d'installation décrite ci-dessus.
Est-ce que PixHelp collecte mes données ? ▾
Absolument pas. PixHelp fonctionne entièrement en local dans ton navigateur. Aucune donnée n'est envoyée à un serveur externe.
L'extension se désactive-t-elle pendant les certifications ? ▾
Oui, c'est automatique. Sur toutes les pages /certifications de Pix, PixHelp se désactive et floute la section concernée avec un message d'avertissement visible.
Comment signaler un bug ou proposer une fonctionnalité ? ▾
Ouvre une issue sur la page GitHub du projet. Les contributions sont les bienvenues !
