PixHelp Prof

Supervisez vos sessions Pix en toute simplicité.

👨‍🏫 Accès direct au tableau de bord

Quand vous êtes sur orga.pix.fr/dashboard, le popup de l’extension affiche un bouton "Tableau de bord prof". Un clic ouvre une interface dédiée.

📊 Visualisation de la session

Consultez la session active, la liste des élèves avec leur statut (actif/inactif) et leur progression. Exportez les données en CSV.

✉️ Actions rapides

Envoyez un message à tous les élèves, ouvrez Pix Orga directement depuis le tableau de bord.

📥 Installation (2 minutes)

  1. Téléchargez le fichier ZIP depuis GitHub.
  2. Décompressez le dossier.
  3. Chrome / Edge : allez dans chrome://extensions/ → activez le mode développeur → "Charger l’extension non empaquetée" et sélectionnez le dossier.
  4. Firefox : allez dans about:debugging → "Ce Firefox" → "Charger un module complémentaire temporaire" → choisissez le fichier manifest.json.
  5. Connectez-vous à Pix Orga et allez sur https://orga.pix.fr/dashboard.
  6. Cliquez sur l'icône PixHelp dans la barre d’extensions : le bouton "Tableau de bord prof" apparaît.
