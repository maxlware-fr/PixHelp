Sécurité & vie privée

Local et sécurisé par conception

PixHelp Prof fonctionne en local dans Firefox et utilise un serveur sécurisé uniquement pour la gestion des sessions. Vos données élèves ne quittent pas le contexte de la session.

📱 Traitement local

L'analyse des exercices et les suggestions pédagogiques sont générées directement dans le navigateur. Aucune consigne ni contenu Pix n'est envoyé à un serveur externe.

🔐 Sessions sécurisées

La fonctionnalité de sessions utilise un serveur sécurisé dédié uniquement à la gestion des codes. Aucune donnée personnelle d'élève n'est stockée.

🔓 Open source

Le code source est entièrement disponible sur GitHub. Auditez, inspectez et contribuez. La transparence est une exigence non négociable.

🚫 Aucun pistage

Pas de publicité, pas de télémétrie, pas de collecte de données d'usage. PixHelp Prof est un outil éducatif, pas un service commercial.

Vos élèves restent vos élèves.

PixHelp Prof ne centralise aucune donnée personnelle. Les codes de sessions expirent automatiquement et ne contiennent aucune information nominative.

Notre philosophie

Un assistant,
pas un substitut.

PixHelp Prof est conçu avec une conviction claire : la pédagogie reste l'affaire de l'enseignant. L'outil n'impose rien, ne décide rien.

Il clarifie les exercices pour que vous puissiez expliquer avec plus d'assurance, sans devoir chercher vous-même.

Il suggère des pistes pédagogiques, mais c'est vous qui choisissez ce qui est adapté à votre classe.

Il s'efface dans l'interface de Firefox, disponible quand vous en avez besoin, invisible sinon.

Il est conçu pour le terrain : rapide, simple, sans courbe d'apprentissage.

PixHelp Prof ne remplace pas le jugement pédagogique de l'enseignant. L'outil sert uniquement d'assistant pour clarifier et structurer l'aide apportée aux élèves.

— Principe de conception de PixHelp Prof
📥 Installation

Disponible sur Firefox

🦊 Uniquement compatible avec Firefox

PixHelp Prof est une extension Firefox. Voici comment l'installer en 4 étapes.

1

Télécharger le ZIP

Récupère la dernière version depuis les releases GitHub et décompresse le dossier.

2

Ouvrir Firefox

Accède à about:debugging dans la barre d'adresse de Firefox.

3

"Ce Firefox"

Clique sur l'onglet "Ce Firefox" dans le panneau de débogage à gauche.

4

Clique sur "Charger un module complémentaire temporaire" et sélectionne le fichier manifest.json.

FAQ

Questions fréquentes

PixHelp Prof fonctionne-t-il sur Chrome ou Edge ? ▾
Non. PixHelp Prof est exclusivement disponible pour Firefox.
Les données de mes élèves sont-elles transmises quelque part ? ▾
Non. L'analyse des exercices se fait entièrement en local. Seule la fonctionnalité de sessions utilise un serveur sécurisé dédié, qui ne stocke que les codes temporaires — aucune donnée personnelle d'élève.
Quelle est la différence avec PixHelp (pour élèves) ? ▾
PixHelp est destiné aux élèves et les aide à trouver les réponses. PixHelp Prof est destiné aux enseignants : il analyse les exercices, propose des pistes pédagogiques et gère les sessions de classe. Les deux outils sont complémentaires mais distincts.
Est-ce gratuit ? ▾
Oui, entièrement gratuit et open-source. Le code source est disponible sur GitHub. Pas d'abonnement, pas de compte obligatoire.
Est-ce une extension officielle de Pix ? ▾
Non. PixHelp Prof est un projet indépendant, non affilié à Pix (GIP Pix). Il est conçu pour fonctionner avec la plateforme Pix sans en faire partie officiellement.
Puis-je utiliser PixHelp Prof pendant une certification officielle Pix ? ▾
PixHelp Prof est conçu pour l'accompagnement pédagogique en classe, pas pour les certifications officielles. Comme toute ressource externe, son usage lors de certifications dépend du règlement en vigueur dans votre établissement.
