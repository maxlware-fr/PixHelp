# PixHelp – Assistant pour la plateforme Pix

PixHelp est une extension navigateur (Chrome / Firefox) qui aide les élèves à trouver des ressources pédagogiques pour réussir les épreuves Pix, et permet aux enseignants de superviser leurs sessions via un tableau de bord dédié.

## ✨ Fonctionnalités
- **Recherche de ressources** (tutoriels, forums, documentation) directement depuis une question Pix
- **Blocage automatique** sur la page de certification (conformément aux conditions d’utilisation)
- **Tableau de bord professeur** : suivi des sessions, liste des élèves, export CSV (version Prof)
- **Paramètres** : activation/désactivation de l’extension

## 📥 Installation
1. Téléchargez le [dernier ZIP](https://github.com/maxlware-fr/PixHelp/releases)
2. Décompressez le dossier
3. **Chrome/Edge** : allez dans `chrome://extensions` → mode développeur → "Charger l'extension non empaquetée"
4. **Firefox** : `about:debugging` → "Ce Firefox" → "Charger un module complémentaire temporaire"

## 🔧 Utilisation
- Sur une épreuve Pix en **Mode Libre**, un bouton flottant apparaît pour chercher des ressources.
- Sur `orga.pix.fr/dashboard`, le popup affiche un bouton pour accéder au tableau de bord professeur.
