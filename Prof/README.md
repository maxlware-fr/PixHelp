# PixHelp Prof – Supervision des sessions Pix

## 🎯 Objectif
Offrir aux enseignants un tableau de bord simple pour suivre les élèves en temps réel lors des sessions Pix.

## 🚀 Fonctionnalités
- **Accès direct** : quand vous êtes sur `orga.pix.fr/dashboard`, un bouton apparaît dans la popup de l’extension.
- **Tableau de bord** : visualisez la session active, la liste des élèves (nom, statut, progression).
- **Export CSV** : exportez les données pour un suivi personnalisé.
- **Actions rapides** : envoyer un message à tous les élèves, ouvrir Pix Orga en un clic.

## 📥 Installation
1. Installez l’extension (voir [README principal](README.md))
2. Connectez-vous à votre compte Pix Orga
3. Rendez-vous sur `https://orga.pix.fr/dashboard`

## 💡 Utilisation
1. Sur `orga.pix.fr/dashboard`, cliquez sur l’icône PixHelp dans la barre d’extensions
2. Cliquez sur **"Tableau de bord prof"**
3. Dans la nouvelle page, consultez la session et les élèves (données simulées pour le moment)
4. Utilisez les boutons pour exporter ou envoyer des messages
