# PixHelp – Aide pour les élèves

## 🎯 Objectif
Trouver rapidement des ressources (tutoriels, forums, documentation) pour vous aider à comprendre les épreuves Pix.

## 🚀 Fonctionnalités
- **Assistant contextuel** : sur une question Pix en Mode Libre, un bouton rond s’affiche. Cliquez, posez votre question, et obtenez des liens vers des tutoriels et des recherches Google ciblées (notamment sur NosDevoirs.fr).
- **Blocage certification** : si vous tentez d’utiliser l’extension pendant une certification, elle se désactive automatiquement et affiche un message d’avertissement.
- **Paramètres** : activez ou désactivez l’extension depuis la popup.

## 📥 Installation
Suivez les instructions générales du [README principal](README.md).

## 💡 Utilisation
1. Allez sur une épreuve Pix (ex: `app.pix.fr`)
2. Assurez-vous que le mode est **"Mode Libre"** (le bouton rond sera coloré)
3. Posez votre question dans la fenêtre de chat
4. Cliquez sur les liens proposés pour accéder aux ressources

## ⚠️ Important
- Ne fonctionne qu’en **Mode Libre** (pour respecter les règles)
- Ne donne pas la réponse directement, mais vous guide vers des ressources pédagogiques
